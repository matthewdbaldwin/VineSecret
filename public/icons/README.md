# PWA Icons

Place the following files here, exported from the VineSecret logo (src/assets/images/VineSecretLogo.webp):

- icon-192.png — 192×192 px, transparent background or solid #09040d
- icon-512.png — 512×512 px, transparent background or solid #09040d

These are referenced by public/manifest.json for the "Add to Home Screen" PWA install prompt.

Tools to generate them:
- Inkscape / Figma / Photoshop — export at the required sizes
- https://maskable.app — preview how the icon looks as a maskable icon
